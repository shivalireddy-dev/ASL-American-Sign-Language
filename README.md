# ASL - American Sign Language Recognition

This project is a machine learning and computer vision application that recognizes American Sign Language (ASL) hand gestures using a webcam. The system detects hand gestures and predicts the corresponding ASL alphabet.

## Features
- Real-time hand gesture detection
- ASL alphabet recognition
- Machine learning based prediction
- User-friendly web interface

## Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-learn
- Flask
- HTML / CSS

## How to Run the Project

1. Clone the repository

git clone https://github.com/shivalireddy-dev/ASL-American-Sign-Language.git

2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py
